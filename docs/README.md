# Documentação

Este diretório concentra os materiais de apoio do benchmark:

- `comparativo.md`: relatório consolidado com métricas e recomendações (gerado por `uv run pdf-extractor evaluate ...`).
- `demo-guide.md`: roteiro rápido com os comandos mais usados de `pdf-extractor demo` (local e via Docker).
- `example-output.md`: saída `.md` de referência para um artigo real.

> Dica: o README principal do projeto descreve todos os comandos possíveis (`extract`, `demo`, `evaluate`), além das opções de execução com UV e Docker. Consulte-o sempre que precisar de um passo a passo completo.

Os arquivos são gerados e atualizados pelos comandos definidos na CLI e versionados para garantir rastreabilidade.
